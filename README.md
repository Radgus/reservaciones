Este proyecto está destinado a usar el MERN Stack.

Pasos para ejecutar el programa

1.- Descargar del repositorio
2.- Instalar dependencias con el comando 
    
    npm install

3.- Ejecutar cualquiera de los siguientes comandos

    npm run start 

    npm start
